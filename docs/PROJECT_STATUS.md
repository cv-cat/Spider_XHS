# 项目状态

最后更新：2026-05-01

## 已完成

- 创建 Web 运营后台基础架构。
- FastAPI 本地后端。
- Vite + React + TypeScript 前端。
- 多账号 Cookie 管理。
- Cookie 校验和短期缓存。
- 发布笔记：图文、视频、话题、地点、可见性。
- 发布任务保存、查看、删除。
- 关键词搜索。
- 关键词监控任务保存、删除、手动执行。
- 搜索结果本地去重保存。
- 单条笔记详情按需加载。
- 媒体代理加载图片和视频。
- 主页查询：当前账号、指定主页链接或 user_id。
- 主页笔记分页查询。
- 账号分析快照。
- 运营概览页。
- 账号级请求限速、失败冷却。
- Docker 单容器部署。
- Docker Compose 一键启动。

## 计划任务

- 发布任务执行器：从任务列表选择任务执行发布，并回写状态。
- 任务队列：支持暂停、恢复、取消、失败重试。
- 关键词监控调度器：按配置间隔自动执行，但必须可暂停、可观察。
- 监控结果筛选：按关键词、账号、时间、是否加载详情筛选。
- 笔记详情缓存：减少重复详情请求。
- 账号分析趋势图：粉丝、互动、近期笔记表现趋势。
- 发布素材管理：素材预览、图片排序、视频封面信息。
- 导出能力：CSV/Excel/JSON。
- 操作日志详情页。
- 更细粒度的风控配置页。

## 每次开发后的更新要求

每次新增或修改功能后，至少更新以下内容：

- `docs/PROJECT_STATUS.md`：补充已完成任务和计划变化。
- `docs/REQUIREMENTS.md`：需求范围变化时更新。
- `docs/DEPLOYMENT.md`：部署方式、端口、环境变量变化时更新。
- `docs/NOTES.md`：新增注意事项、风险或约束时更新。
- `RISK_CONTROL.md`：风控策略变化时更新。
